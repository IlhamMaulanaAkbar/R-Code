# Aplikasi R-CODE (Ruang Koding)
## Ringkasan
Aplikasi R-CODE (Ruang Koding) adalah sebuah aplikasi yang bisa menampilkan informasi tentang usaha yang ditawarkan seperti course, partners, dan contact dari perusahaan.

## Fitur Aplikasi
- Halaman Utama 
- Halaman Admin
- Halaman Login

## Dibuat Dengan :
- PHP
- Boostrap
